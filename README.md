# LOLsmurf-trollhoihoi
🐾 LoL Sumatoro Hoihoy (Smurf & Troll Detector)
Developed by Sakku at Sakuramori Lab, this specialized tool is designed to protect streamers and their communities from the toxicity and imbalance caused by smurfs and trolls in League of Legends.
🐱 Background
Created specifically to ensure a stress-free environment for streamers like Vanilla (Vanilla Maple Project). In the chaotic world of LoL, this tool acts as your personal "guard cat," identifying players with suspicious stats using the Riot Games API.
🛠️ Key Features
• Smart Smurf Detection: - Analyzes Win Rate, KDA, CS/Min, Vision Score, and Account Level.
• Custom algorithm to identify high-rank players hiding in lower tiers.
• Troll & Chat Analysis:
• Real-time detection of NG words, spamming, and toxic sentiment.
• Scoring system for disruptive behavior with optional automated "healing" replies.
• Multi-Region Support: Supports JP1, NA, KR, EU, and other major Riot servers.
• Discord Integration: Sends real-time alerts via Discord Webhooks.
• Modern GUI: A user-friendly control panel built with Python's Tkinter.
🏗️ Tech Stack
• Language: Python 3.x
• Framework: Tkinter (GUI)
• API: Riot Games API (Match-v5, Summoner-v4, League-v4)
• Networking: requests for robust API handling
🚀 Getting Started
1.	Obtain an API Key from the Riot Developer Portal.
2.	Enter your API Key and preferred Region in the Settings tab.
3.	Input a Summoner Name and click "Analyze"!
4.	Add players to the "Watch List" for continuous monitoring.
